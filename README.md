# FitTrack - Трекер тренировок

Современное веб-приложение для отслеживания тренировок с интуитивным интерфейсом и полезными функциями для спортсменов.

## Возможности

### Аутентификация

- Регистрация и вход в систему
- Безопасное хранение сессии

### Управление тренировками

- Создание новых тренировок с указанием даты и описания
- Редактирование даты и описания существующих тренировок
- Удаление тренировок
- Копирование тренировки на текущую дату с пометкой "(Копия)"
- Визуальные индикаторы статуса: "Сегодня", "Запланировано", "Завершено"
- Сортировка по дате (недавние первыми)

### Упражнения

- Добавление упражнений с параметрами: название, повторения, вес, время отдыха
- Редактирование и удаление упражнений
- Встроенный таймер отдыха с настраиваемым временем из упражнения

### Профиль и статистика

- Серия тренировок (streak)
- Тренировок за неделю с прогресс-баром
- Общий поднятый вес (в тоннах)
- Любимое упражнение
- Максимальный вес

- Node.js 18+

## Установка

1. Клонируйте репозиторий:

```bash
git clone <repository-url>
cd fittrack
```

2. Установите зависимости:

```bash
npm install
```

3. Запустите приложение:

```bash
npm run dev
```

4. Откройте [http://localhost:3000](http://localhost:3000)

## API Endpoints

Приложение ожидает следующие endpoints на backend:

### Аутентификация

- `POST /auth/register` - Регистрация
- `POST /auth/login` - Вход
- `GET /auth/check` - Проверка авторизации
- `POST /auth/logout` - Выход

### Тренировки

- `GET /trainings` - Список тренировок
- `POST /trainings` - Создать тренировку
- `PUT /trainings/:id` - Обновить тренировку
- `DELETE /trainings/:id` - Удалить тренировку

### Упражнения

- `GET /trainings/:id/exercises` - Список упражнений
- `POST /trainings/:id/exercises` - Добавить упражнение
- `PUT /exercises/:id` - Обновить упражнение
- `DELETE /exercises/:id` - Удалить упражнение
